### 환경 변수 설정



시작 => "시스템 환경 변수"검색 => 시스템 환경 변수 편집



![image-20200217102916062](images/image-20200217102916062.png)



환경 변수 => 시스템 변수 => 새로 만들기



변수 이름 : MAVEN_HOME

변수 값 : 압축 해제한 경로 (C:\Users\TTak\apache-maven-3.6.3)



시스템 변수 목록에서 Path를 찾아 더블 클릭 => 새로 만들기

%MAVEN_HOME%\bin 추가
